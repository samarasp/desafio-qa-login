# Desafio QA — Sistema de Login

Projeto desenvolvido para prática dos fundamentos de Qualidade de Software.

## Conteúdo do desafio

- Fluxo de trabalho do desenvolvimento
- Ciclo de vida do bug
- User Stories
- Mind-map
- Casos de teste step-by-step
- Casos de teste BDD

## Objetivo

Demonstrar conhecimentos básicos sobre documentação e processos de teste de software utilizando um sistema simples de login.
